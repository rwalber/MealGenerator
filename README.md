# MealGenerator
Feeling hungry? Get a random meal by clicking below!

##### Technologies used: ReactJS and NodeJS.

![Screenshot](assets/first.png)

![Screenshot](assets/second.png)

## Build project

##### Run back-end
```
backend/

Install node modules:
npm install

Start
yarn dev
```

##### Run front-end
```
frontend/

Install node modules:
npm install

Start
yarn start
```
